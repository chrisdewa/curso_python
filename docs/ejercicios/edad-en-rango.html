7.1. Edad en rango#

Para un estudio ficticio se requiere una muestra aleatoria de 30 individuos, tu objetivo es generar en forma aleatoria las ids y utilizarlas para seleccionar los sujetos del conjunto de datos propuesto.

7.1.1. Conocimiento previo#

En la librería estándar de python existe el módulo random que permite realizar muchas de las operaciones aleatorias más comunes.

Para este ejercicio requieres una función que permite el muestreo aleatorio de una población, revisa la documentación de random.
